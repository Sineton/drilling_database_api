## [1.0.0] - 2026-04-20

### Added

- Initial standalone publication of the FastAPI-based drilling database API.
- Support for wells, wellbores, events, LAS import, Excel import, supervisor journal import, supervisor event synchronization, analytics, datasets, and markup import.
- English repository `README.md` with setup instructions, endpoint overview, and usage examples.
- Safe `.env.example` template and repository-level `.gitignore` for local secrets and non-source artifacts.

### Changed

- Replaced hardcoded database credentials in the default configuration with safe local placeholder values for GitHub publication.
